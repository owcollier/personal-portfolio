News Spot
is a responsive wep application which allows users to search for top news headlines/stories, share news stories externally on a variety of social networking platforms, and curate stories internally via customizable collections
  • Front end built with React & Redux
  • Back end built with Node.js & Express
  • Features user authentication utilizing bcrypt for securing user passwords & Passport.js for local authentication strategies
  • Features oAuth supporting Google & Facebook authentication strategies
  • All app data stored using MongoDB & queried using Mongoose
