News Spot

https://newsspot.netlify.com/

News Spot is a responsive news search and curation app using the open-source News API. I worked with a team to design and develop News Spot, and we had three weeks to flex our developer skills and churn out a product with a rich suite of functionality.

We managed to create an app with user authentication featuring both Google and Facebook oAuth, a robust search functionality pulling from hundreds of news sources, a curation system which allows our user to save news stories to personalized collections/share news stories externally, and a global activity feed reflecting user interactions across the entire app.

  • Front end built with React & Redux
  • Back end built with Node.js & Express
  • Features user authentication utilizing bcrypt for securing user passwords & Passport.js for local authentication strategies
  • Features oAuth supporting & Facebook authentication strategies
  • All app data stored using MongoDB & queried using Mongoose

French Flash Cards

https://frenchflashcards.netlify.com/

French Flash Cards is a language learning application that presents the user with a set of French vocabulary flash cards for "spaced repetition" based practice. The concept of spaced repetition learning asserts that a smarter approach to retaining knowledge is to review what you have a weaker grasp on more often, and review what you have a stronger grasp on less often. In the context of flashcards, this means that cards the user has answered incorrectly should reappear at a greater frequency than cards the user has answered correctly.

My approach to implementing the spaced repetition pattern is an algorithm embedded server-side that handles user input during the exercises and addresses the data structure of the flash cards accordingly. The flashcards are arranged in linked list data structure, with each possessing a "memory strength" property in the database model. When a user answers a flashcard correctly, the value for a card's memory strength increases and the algorithm pushes the position of that card further back in the linked list. If the flashcard is answered incorrectly, the card's memory strength is reset to a default value and the flash card will reappear closer to the head of the linked list, thus increasing the frequency of that flashcard in the learning exercise.

  • Front end built with React & Redux
  • Back end built with Node.js & Express
  • Features user authentication utilizing bcrypt for securing user passwords & Passport.js for local authentication strategies
  • Spaced repetition based learning facilitated via an algorithm which tracks user answers in a singly linked list and adjusts position of flash cards in the deck according to users' demonstrated mastery of each flash card
  • All app data stored using MongoDB & queried using Mongoose
